# 广州地铁 TSG18 型受电弓

此处提供了由**Max 温焰**建模的广州地铁 TSG18 型受电弓的 **blend**文件 以及 **obj 文件** 注意：obj 文件不支持统一升降弓功能（该功能使用 Blender 实现）

## 介绍

TSG18 型受电弓最早使用于庞巴迪 Movia 456 型列车(A2)，直到如今依然在使用
